# MyAnimeList

MyAnimeList is an Android application that showcases a list of anime, fetched from an external API. The project is built with modern Android development practices and a robust set of libraries.

## Features

*   Browse a list of anime.
*   View anime details.

## Technologies Used

*   **Kotlin:** The primary programming language for the application.
*   **Android Jetpack:**
    *   **ViewModel:** To store and manage UI-related data in a lifecycle-conscious way.
    *   **LiveData:** To build data objects that notify views when the underlying database changes.
    *   **Navigation Component:** To handle all navigation within the app.
    *   **Room:** For local database storage.
*   **Dagger Hilt:** For dependency injection.
*   **Retrofit:** A type-safe HTTP client for Android and Java.
*   **Gson:** A Java serialization/deserialization library to convert Java Objects into JSON and back.
*   **Glide:** An image loading and caching library for Android.
*   **Android YouTube Player:** A library for embedding and controlling YouTube videos in the app.
